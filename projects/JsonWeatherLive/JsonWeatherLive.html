JSON WEATHER LIVE

Interconnected, location is meaningless // The warming sun of Venice. The blissful breeze of Himalaya. The wrath of Katrina. Unprecedented millennial droughts // All in one, they combine in an incoherent and void data swarm // But once brought to perception, sun is rumble. Heat is chromatic. Mist is rhythmic // Global is local

This Max for Live device is an update to the JSON Weather device of the Connection Kit made by Ableton. It allows the control of any Ableton Live parameter based on real-time weather data at any desired location.

Made for Max 6.1 and above
